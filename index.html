SMEditor

An open source tool to view and edit StepMania charts.

All Downloads

Features

Automatic Parity Detection

SMEditor automatically determines which foot to use for each note, making pad charting easier.

Custom Scripting

Automate tasks by using the built-in scripting engine.

Audio Filtering and Tempo Detection

Easily sync your charts with the built-in EQ and tempo detection tools.

Full Customization

Change the editor's theme, noteskins, and key binds to your liking.

Help Guide

Learn how to use SMEditor with the built-in help guide, which contains tutorials and documentation.
